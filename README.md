# Rogue-Lineage-TD
A feature-rich, 2D Tower Defense game built in Python using the pygame library. This project features modular tower systems, enemy pathfinding, and a branching upgrade system.

# Features
Branching Upgrades: Choose between two unique paths (Orderly or Chaotic) for each tower type to specialize your defense.
Dynamic Tower System: Includes specialized towers like Warrior, Scholar, Cadence, Thief, and Brawler, each with unique mechanics inspired by Rogue Lineage.
Multimedia Integration: Incorporates audio cues and video elements using pygame and cv2 (OpenCV).

# Prerequisites
To run this game, you must have Python installed. You will also need to install the following dependencies:
pip install pygame opencv-python numpy

# How to Run
1. Clone this repository to your local machine:
git clone https://github.com/alliandev/Rogue-Lineage-TD.git
2. Navigate to the project folder:
cd Rogue-Lineage-TD
3. Launch the game:
python main.py
